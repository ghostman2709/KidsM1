🚀 Como Funciona o HiperBoot

O cliente faz o pedido → O boot mostra o produto → O cliente compra e envia o comprovante → A API verifica automaticamente se o valor entrou no sistema.

✅ Se confirmado: o boot libera o produto. Para produtos físicos, ele aciona automaticamente o motoboy ou entregador de confiança com as instruções de retirada e entrega. Para produtos digitais, o link de acesso é liberado instantaneamente.
