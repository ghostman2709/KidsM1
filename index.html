Módulo 1 – O Que é o Dinheiro?

História: Joãozinho e a Moeda Perdida

Joãozinho encontrou uma moeda dourada no caminho da escola. Ela brilhava como o sol e parecia mágica! Ele ficou tão feliz que correu para mostrar ao avô. O vovô explicou:

— Joãozinho, o dinheiro é como essa moeda: só tem valor quando sabemos usá-lo com sabedoria.

— Mas para que serve o dinheiro? — perguntou João.

— Para comprar coisas que precisamos, guardar para o futuro, ou até ajudar outras pessoas. O dinheiro é uma ferramenta, não um brinquedo.

Explicação

O dinheiro é um meio para trocar coisas, guardar para um momento especial e construir sonhos. Ele não é mágico, mas pode ajudar a realizar coisas incríveis quando usado com cuidado.

Frase de Impacto

“O dinheiro é um bom servo, mas um péssimo mestre.” — Francis Bacon

Reflexão

Se você tivesse uma moeda mágica, o que escolheria fazer com ela: gastar, guardar ou doar? Por quê?
